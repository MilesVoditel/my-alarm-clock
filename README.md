# my-alarm-clock

What I did:
I designed a alarm clock that heavily (heavily=somewhat) resembles a bomb defusal tool, based off the one in CSGO

Purpose of Build:
I built this alarm clock to replicate my feeling when finding this project (as the due date was the next day), "Time is running out". With inspiration from the game csgo (and technically csgo 2), this alarm clock will let you relive the joys of stressful bomb defusals first thing in the morning, right after you wake up.

Problems I Faced:
One of the main problems I faced were time constraints. I only learned about hack club as a whole ~36 hours before the deadline for BLARE, so completing this project was definitely a scramble (Liam was a lifesaver and should 100% be kept on the team after his gap year in case of emergencies). The other main problem I faced was having very little applicable experience in coding, with the main chunk of coding I've done being based in processing (thanks sophmore level coding class). The only way I was able to get this done in time was due to the resources and help provided by Liam and the other participants in BLARE.
